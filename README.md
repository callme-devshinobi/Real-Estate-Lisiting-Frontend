# 🏡 MyProperty - SPA Property Listing Website

## 📌 Overview

MyProperty is a modern single-page application (SPA) for browsing real estate listings. Built using HTML, CSS, JavaScript, and Bootstrap, it delivers a clean, responsive, and user-friendly interface for exploring properties.

---

## 🚀 Features

* Responsive design for mobile, tablet, and desktop
* Hero section with interactive search bar
* Property search and filtering functionality
* Featured property listings
* Dynamic UI updates using JavaScript
* Clean and modern layout using Bootstrap
* Reusable components for scalability

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)
* Bootstrap 5

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/callme-devshinobi/Real-Estate-Lisiting-Frontend.git
```

2. Navigate to the project folder

```
cd Real-Estate-Lisiting-Frontend
```

3. Open `index.html` in your browser
